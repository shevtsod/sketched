# What is This?

Sketched is an online multiplayer web game about drawing things from memory that
you can host yourself!

It was inspired by this video by streamer
[39daph](https://www.youtube.com/@39daph) about drawing Pokémon from memory and
then comparing against the originals:

[![Drawing 151 Pokémon From Memory](https://img.youtube.com/vi/7GQDIlrarWg/0.jpg)](https://www.youtube.com/watch?v=7GQDIlrarWg)
